\section*{How to Use This Guide}
\addcontentsline{toc}{section}{How to Use This Guide}
\begin{itemize}[leftmargin=*]
  \item Work through the \textbf{Homework (Problems)} section first to attempt each problem set under test-like conditions.
  \item Use the \textbf{Solutions} section to check your work and study the provided steps from the official solution sets.
  \item Keep the \textbf{Cheat Sheets} section open while reviewing formulas, definitions, and quick-reference identities.
  \item The \textbf{Resource Appendices} section contains supplementary references and the course text to reinforce concepts.
\end{itemize}
